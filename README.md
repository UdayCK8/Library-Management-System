# Library-Management-System
The Library Management System is a web-based application that helps manage books, users, and transactions efficiently. Admins can add, update, or delete books, while users can register, browse, and borrow books. It automates library operations for faster and smarter management.  Built using HTML, CSS, PHP, and MySQL, the project focuse


🔹 Key Features:

User login and role-based access (Admin/User)

Book issue, return, and search functionalities

Real-time database updates using MySQL

Clean, responsive design for better usability
